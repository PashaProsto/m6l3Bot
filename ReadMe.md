# Многофункциональный Telegram бот

Бот для повседневных задач: заметки, генератор паролей, игры и погода.

## Возможности

### Заметки
- Создание текстовых заметок
- Просмотр списка заметок
- Удаление заметок по ID

Все заметки сохраняются в базе данных.

### Генератор паролей
- Простой режим - случайные символы
- Сильный режим - заглавные и строчные буквы, цифры, спецсимволы
- Проверка надежности пароля
- Сохранение паролей в личное хранилище
- Просмотр и удаление сохраненных паролей

Длина пароля: от 4 до 64 символов.

### Крестики-нолики
- Режим против бота
- Режим для двух игроков

### Погода
- Текущая погода по названию города
- Сохранение города командой /setcity
- Быстрый просмотр погоды в сохраненном городе

## Команды

- /start - запуск бота и главное меню
- /setcity - сохранить город для погоды

## Управление

Все функции доступны через кнопки главного меню:
- Заметки - управление заметками
- Пароли - генерация и хранение паролей
- Крестики-нолики - выбор режима игры
- Погода - погода по городу
- Моя погода - погода в сохраненном городе
- Мои пароли - просмотр сохраненных паролей
